# 使用 Vite 构建 React 应用教程

## 1. 安装依赖

```bash
node -v
npm -v
```

## 创建 React + Vite 项目

```bash
# 选 React + TypeScript
npm create vite@latest my-react-app
```

进入项目并安装依赖

```bash
cd my-react-app
npm install
```

启动开发服务器

```bash
npm run dev
```

## 构建生产环境版本

```bash
npm run build
```

生成的文件在：

```bash
dist/
```

本地预览构建结果：

```bash
npm run preview
```
